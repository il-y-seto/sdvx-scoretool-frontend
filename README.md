# sdvx-scoretool-frontend

> My impressive Nuxt.js project

## はじめかた

```bash
# .envファイルをコピー
$ cp .env.example .env

# nuxtのコンテナをビルドして立ち上げ、パッケージの取得等も行う
$ make dev-install

# コンテナを立ち上げる
$ make up

# npm run devする
$ make start

# generate static project(まだうごかない)
$ yarn generate
```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
